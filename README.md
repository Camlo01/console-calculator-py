# Console Calculator Python

This application was created to practice the Python language, methods, syntax, and more. The purpose of this project is to improve programming skills through the implementation of a console calculator.

## Features

- Basic operations: addition, subtraction, multiplication, and division.
- Error handling: input validation and management of common errors.
- Simple user interface based on the console.
- Modular and organized code for easy understanding and maintenance.

## Requirements

- Python 3.x
